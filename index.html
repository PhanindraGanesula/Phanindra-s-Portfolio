Weather & Travel Guide Application

Developed a dynamic weather application using HTML, CSS, and JavaScript that provides real-time weather updates through an API. The app displays key metrics like humidity and wind speed. A standout feature is its integration of travel information: searching for a location dynamically updates the background with a picture of a famous landmark from that area and provides travel tips, including the best local food to try and the ideal time to visit.
